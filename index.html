What is a string?

A string
is a sequence of characters used to represent text in programming. It can include letters, numbers, symbols, and spaces.

greeting = "Hello, World!"

What is a Data type?

A data type
is a classification that specifies the type of data that a variable or expression can hold in a programming language. It defines the operations that can be performed on the data and the way the data is stored in memory.

Common Data Types : Interger, String, float, Boolean , ArrayList

What is a OOP?

Object-Oriented Programming (OOP)
is a programming paradigm that structures software design around objects, which are instances of classes.

What is a Boolean?

A Boolean
is a data type that represents one of two possible values : true or false.

What is a Array?

An array
is a data structure that stores a fixed-size sequence of elements of the same type in contiguous memory locations. Arrays allow for efficient access to elements using an index.

What is a Loop?

A loop
is a control structure used in programming to repeat a block of code multiple times based on a condition or a specified number of iterations.

Types of Loops: For Loop, While Loop, Do-While Loop

What is a variable?

A variable is a named storage location in programming used to hold data that can be referenced and manipulated throughout a program.

What is frontend developer do?

A frontend developer
is a type of developer that is work on User interface part

What is a Backend developer do?

Backend dev
focuses on the server-side, which involves creating and managing the logic, database interactions, and server configuration.

What is a Devop devloper do?

Devop
focuses on the practices and tools that bridge the gap between software development (Dev) and IT operations (Ops)

What is a QA Engineer do?

A QA (Quality Assurance) Engineer
ensures that software products meet specified quality standards before they are released

What is a HTML?

HTML (HyperText Markup Language)
is the standard language used to create and structure content on the web.

What is a CSS?

CSS (Cascading Style Sheets)
is a language used to describe the presentation and layout of HTML elements on a webpage.

What is a JavaScript?

JavaScript
is a high-level, interpreted programming language that is primarily used to create dynamic and interactive effects on webpages.

What is an API?

API stands for Application Programming Interface.
It is a set of rules and protocols that allows different software applications to communicate with each other.

What is a SQL?

SQL stands for Structured Query Language
It is a standard programming language used to manage and manipulate relational databases.
